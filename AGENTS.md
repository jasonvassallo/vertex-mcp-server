# AGENTS.md

Guidance for OpenAI Codex (and other AGENTS.md-aware tools) reviewing or editing
this repository.

If a `CLAUDE.md` exists, read it first — it is the canonical architecture and
conventions doc. For reviews, focus on correctness/bugs, security, performance,
error handling, and test coverage. Be concrete and concise; skip stylistic nits.
Flag changes that weaken existing safety checks or silently swallow errors.
