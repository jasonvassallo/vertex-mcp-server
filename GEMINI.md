# GEMINI.md

Context for the Gemini CLI (`google-github-actions/run-gemini-cli`) when reviewing
this repo. If a `CLAUDE.md` exists, read it first — it is the canonical
architecture/conventions doc.

For PR reviews, focus on: correctness and likely bugs, security, performance,
error handling, and test coverage. Be concrete and line-specific; skip purely
stylistic nits. Flag anything that weakens existing safety checks or silently
swallows errors.
